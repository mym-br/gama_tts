
GamaTTS:Editor
==============

This is the editor for the articulatory database in GamaTTS.

GamaTTS:Editor is an approximate clone of Monet, an editor in the [Gnuspeech][]
project.

[Gnuspeech]: http://www.gnu.org/software/gnuspeech/

External code
