
- Requirements:

  - CMake.

  - A C++ compiler with C++17 support.
    The following compilers have been tested:
    - GCC 10.2      (Linux+GNU x86_64).
    - LLVM/Clang 11 (Linux+GNU x86_64).

  - The current directory must be named "gama_tts".

- Build:

  - Linux+GNU - GCC:

    mkdir ../gama_tts-build
    cd ../gama_tts-build
    cmake -D CMAKE_BUILD_TYPE=Release ../gama_tts
    cmake --build .

  - Linux+GNU - Clang:

    mkdir ../gama_tts-build
    cd ../gama_tts-build
    CXX=clang++ cmake -D CMAKE_BUILD_TYPE=Release ../gama_tts
    cmake --build .

- Test (Linux+GNU):

  - Execute in the directory "gama_tts-build":

    echo "Hello world." | ./gama_tts tts \
      ../gama_tts/data/voice/english/5_male test.wav

  - Play the file test.wav.
